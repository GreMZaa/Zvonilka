- [x] **1.2** Реализовать создание комнаты
  - Генерировать уникальный `roomId` (через `utils.generateId()`)
  - Определить роли: `caller` (кто звонит) и `callee` (кто отвечает)
- [x] **1.3** Реализовать отправку сигналов
  ```js
  async function sendSignal(roomId, type, payload, sender) {
    await supabase.from('signaling').insert({ room_id: roomId, type, payload, sender });
  }
  ```
- [x] **1.4** Реализовать подписку на входящие сигналы
  ```js
  function subscribeToRoom(roomId, role, onSignal) {
    supabase.channel(`room:${roomId}`)
      .on('postgres_changes', {
        event: 'INSERT',
        schema: 'public',
        table: 'signaling',
        filter: `room_id=eq.${roomId}`
      }, (payload) => {
        if (payload.new.sender !== role) onSignal(payload.new);
      })
      .subscribe();
  }
  ```
- [x] **1.5** Реализовать очистку — отписку от канала при завершении звонка
- [x] **1.6** Экспортировать API модуля: `initSignaling()`, `createRoom()`, `joinRoom()`, `sendSignal()`, `cleanup()`

### Критерий готовности
✅ Открыть два окна браузера → одно создаёт комнату → другое подключается → JSON-сообщения доходят в обе стороны за <1 сек. Проверить через `console.log`.
