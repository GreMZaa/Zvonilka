# 📦 Дистрибуция и Установка приложения «Звонилка»

Этот документ содержит пошаговые руководства по установке приложения для мамы (на Android или iOS) и сборке установочного APK-файла.

Сайт приложения опубликован по адресу:  
🔗 **[https://gremzaa.github.io/Zvonilka/](https://gremzaa.github.io/Zvonilka/)**

---

## 📱 Инструкция для мамы (iOS / iPhone)

Для работы приложения на iPhone не нужно ничего скачивать из App Store. Установка занимает 1 минуту через браузер Safari:

1. Открой стандартный браузер **Safari** на iPhone.
2. Перейди по ссылке: **`https://gremzaa.github.io/Zvonilka/`**
3. Нажми кнопку **«Поделиться»** (иконка квадрата со стрелкой вверх внизу экрана).
4. Прокрути меню вниз и выбери пункт **«На экран "Домой"»** (или *Add to Home Screen*).
5. Нажми **«Добавить»** в правом верхнем углу.
6. На рабочем столе появится иконка **«Звонилка»**. Запускай приложение только через эту иконку (оно будет работать во весь экран без рамок браузера).

## 🤖 Инструкция для мамы (Android / Установка APK)

Если у мамы телефон на Android (Samsung, Xiaomi, Huawei и др.), рекомендуется установить приложение как полноценный APK-файл.

### Шаг 1. Сборка APK-файла (Инструкция для разработчика)
Если у вас ещё нет APK-файла, соберите его за 2 минуты:
1. Перейдите на сайт **[PWABuilder.com](https://www.pwabuilder.com/)**.
2. Вставьте ссылку на ваше приложение: `https://gremzaa.github.io/Zvonilka/` и нажмите **Start**.
3. Сервис протестирует ваш PWA-манифест (все тесты должны быть зелеными).
4. Нажмите кнопку **Package for Store** в секции **Android**.
5. Нажмите **Generate** (настройки по умолчанию оптимальны).
6. Скачайте готовый архив. Внутри будет файл с расширением `.apk` (например, `app-release-unsigned.apk` или `app-release.apk`).
7. Отправьте этот файл маме в любой мессенджер (Telegram, WhatsApp) или загрузите на Яндекс.Диск / Google Drive.

### Шаг 2. Установка APK на телефоне мамы
1. Скачайте присланный APK-файл на телефон.
2. Запустите его установку.
3. Если телефон выдаст предупреждение: *«В целях безопасности ваш телефон блокирует установку приложений из неизвестных источников»*:
   - Нажмите кнопку **Настройки** прямо в этом окне.
   - Включите переключатель **Разрешить установку из этого источника** (или разрешите для Telegram/WhatsApp/Браузера).
   - Вернитесь назад и нажмите **Установить**.
4. После завершения установки иконка **«Звонилка»** появится на главном экране.

## 📞 Инструкция: Как совершить первый звонок

Для установки WebRTC соединения оба устройства должны находиться в одной "комнате" сигналинга.

1. **Звонящий (Вы или Мама):**
   - Откройте приложение «Звонилка».
   - Разрешите доступ к микрофону при первом запуске.
   - Нажмите на большую кнопку **«Позвонить маме»**.
   - Появится надпись *«Соединение...»*, и вверху экрана отобразится **8-значный код комнаты** (например, `4f8g9h2k`).
   - Скопируйте или продиктуйте этот код второй стороне.

2. **Принимающий звонок:**
   - Откройте приложение «Звонилка».
   - Разрешите доступ к микрофону.
   - Введите полученный **код комнаты** в текстовое поле ввода.
   - Нажмите кнопку **«Войти»**.
   - Экран переключится в состояние *«Входящий вызов (Мама звонит!)»*.
   - Нажмите круглую зеленую кнопку **«Ответить»**.

3. **Связь установлена!**
   - На обоих экранах загорится статус **🟢 Разговор** и запустится таймер длительности звонка.
   - Голос будет передаваться напрямую (Peer-to-Peer) в зашифрованном виде.
   - Для завершения разговора в любой момент нажмите круглую красную кнопку **«Сбросить»**.
