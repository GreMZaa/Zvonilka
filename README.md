# Звонилка — WebRTC PWA

Бесплатное, устойчивое к блокировкам приложение для голосовых звонков.

## Быстрый старт

1. Заполнить `src/js/config.js` своими ключами Supabase
2. Открыть `index.html` в браузере (или через Live Server)
3. Для деплоя — запушить в GitHub Pages

## Структура проекта

```
Dialer/
├── index.html              # Главная страница (точка входа)
├── manifest.json           # PWA-манифест (для установки как приложение)
├── sw.js                   # Service Worker (офлайн-кеширование)
│
├── src/                    # Исходный код
│   ├── js/                 # JavaScript-модули
│   │   ├── app.js          # Точка входа — инициализация всех модулей
│   │   ├── config.js       # Конфигурация (Supabase, ICE, таймауты)
│   │   ├── webrtc.js       # WebRTC — PeerConnection, медиа, ICE
│   │   ├── signaling.js    # Сигналинг — обмен offer/answer через Supabase
│   │   ├── ui.js           # Интерфейс — кнопки, статусы, анимации
│   │   └── utils.js        # Утилиты — генератор ID, логирование
│   │
│   └── css/                # Стили
│       ├── styles.css      # Основные стили и дизайн-токены
│       └── animations.css  # Keyframes и анимации
│
├── assets/                 # Статические ресурсы
│   ├── icons/              # Иконки приложения (PWA)
│   │   ├── icon-192.png    # Иконка 192x192
│   │   └── icon-512.png    # Иконка 512x512
│   ├── sounds/             # Звуки (рингтон, уведомления)
│   └── images/             # Изображения (если понадобятся)
│
├── docs/                   # Документация
│   └── ARCHITECTURE.md     # Описание архитектуры
│
└── .gemini/                # Контекст для AI-ассистента
    └── CONTEXT.md          # PRD проекта
```
